Bienvenido a Lossless DC

Lossless DC es un programa diseñado para comprimir y descomprimir archivos de manera eficiente, sin pérdida de información. En la ventana principal, encontrarás tres apartados principales:

  • Inicio: Aquí puedes realizar las tareas de compresión y descompresión de archivos.
  • Tutorial: La sección que estás leyendo, que te guiará paso a paso en el uso del programa.
  • Documentación: Información técnica adicional sobre los algoritmos y el funcionamiento del programa.

Compresión de Archivos

Lossless DC permite comprimir archivos de texto plano sin perder ninguna información. Para comprimir un archivo:

  1. Selecciona el archivo de texto plano: Haz drag and drop (arrastrar y soltar) el archivo de texto en el área de Compresión, o haz clic en el campo para abrir una ventana de selección de archivos.
  2. Iniciar compresión: Después de cargar el archivo, aparecerá el botón "Comprimir". Haz clic en él para comenzar el proceso de compresión.
  3. Datos de la compresión: Durante y después del proceso de compresión, Lossless DC te mostrará algunos datos clave sobre el archivo:
    • Porcentaje comprimido: Muestra la reducción de tamaño lograda.
    • Tamaño original: El tamaño del archivo antes de ser comprimido.
    • Tamaño comprimido: El tamaño final del archivo después de la compresión.

Descompresión de Archivos

La descompresión te permite recuperar los archivos originales a partir de los comprimidos. Para descomprimir un archivo con Lossless DC, necesitarás dos archivos:

  1. Selecciona los archivos necesarios:
    • Archivo .json: Este archivo contiene la codificación que se utilizó para comprimir el archivo.
    • Archivo .bin: El archivo comprimido que deseas descomprimir.
    Puedes cargar ambos archivos utilizando la opción de drag and drop o hacer clic en los campos respectivos para seleccionar los archivos.
  2. Iniciar descompresión: Una vez que hayas cargado ambos archivos, aparecerá el botón "Descomprimir". Haz clic en él para iniciar el proceso de descompresión.
  3. Datos de la descompresión: Después de completar la descompresión, Lossless DC te proporcionará detalles sobre el proceso, tales como:
    • Tamaño del archivo descomprimido: El tamaño del archivo después de la descompresión.
    • Tiempo de descompresión: Cuánto tiempo ha tardado en realizarse el proceso.

Consejos y Recomendaciones

  • Archivos de Texto Plano: Asegúrate de que el archivo que subes para compresión esté en formato de texto plano, como .txt.
  • Archivos .json y .bin: Para la descompresión, necesitas ambos archivos: el archivo .json con la codificación y el archivo .bin con los datos comprimidos. La ausencia de cualquiera de estos archivos impedirá que el proceso de descompresión se complete correctamente.
  • Tiempo de Procesamiento: El tiempo de compresión y descompresión puede variar dependiendo del tamaño de los archivos y la complejidad de los datos. Ten paciencia mientras se procesa.
